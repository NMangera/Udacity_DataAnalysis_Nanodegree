# Udacity_DataAnalysis_Nanodegree

This repository contains projects for Udacity's Data Analyst Nanodegree.


## Prerequisites

You'll need to install:

Python (3.x or higher)
Jupyter Notebook

Numpy
Pandas
Matplotlib
Seaborn
additional libraries defined in each project.

Recommended:
Anaconda
